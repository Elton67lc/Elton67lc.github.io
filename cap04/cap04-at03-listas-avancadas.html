Seção 2 — Top Tecnologias do Mercado

  1. Bootstrap

    Framework CSS que acelera o desenvolvimento de interfaces responsivas e elegantes com componentes prontos.

  2. Node.js

    Ambiente de execução que permite utilizar o JavaScript no lado do servidor (backend), escalando aplicações.

  3. JavaScript

    A alma da web moderna. Responsável por toda a lógica dinâmica e interatividade do lado do cliente.

  4. CSS

    Linguagem de estilos encarregada do design, das cores, das animações e do layout visual das páginas.

  5. HTML

    A fundação de tudo. Define a estrutura semântica e a organização do conteúdo para navegadores e buscadores.

Seção 3 — Dicionário de Termos Técnicos

HTML
Linguagem de marcação para estruturar documentos na web.
Sigla para HyperText Markup Language.
CSS
Folhas de Estilo em Cascata, usadas para definir a apresentação visual.
JavaScript
Linguagem de programação de alto nível, interpretada e voltada a objetos.
DOM
Document Object Model: uma interface que representa o documento HTML como uma árvore de objetos.
Permite que linguagens como JS acessem e manipulem a estrutura da página.
API
Application Programming Interface
Conjunto de rotinas e padrões que permitem que diferentes softwares se comuniquem.
HTTP
Hypertext Transfer Protocol
O protocolo fundamental para a troca de dados e navegação na rede mundial de computadores.
URL
Localizador Padrão de Recursos; o endereço digital único de um arquivo ou página.
DNS
Domain Name System: o sistema que converte nomes amigáveis (como google.com) em endereços IP.
